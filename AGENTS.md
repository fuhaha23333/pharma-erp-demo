# AGENTS.md

## 项目背景
本项目是一个小型药品批发 ERP，目标是先完成 MVP：
客户首营、供应商首营、商品首营、采购验收入库、批号库存、销售出库、库存流水。

## 技术栈
- Java 17
- Spring Boot
- MyBatis-Plus
- MySQL
- Sa-Token
- Vue3 + Element Plus

## 开发原则
1. 不做微服务。
2. 不做复杂工作流引擎，第一版只做固定审批状态流。
3. 所有业务单据采用主表 + 明细表结构。
4. 库存必须按：药品 + 批号 + 效期 + 仓库 + 货位 + 质量状态 管理。
5. 每次库存变化必须生成 stock_movement 库存流水。
6. 所有金额使用 BigDecimal。
7. 所有状态使用枚举，不要直接写魔法字符串。
8. 重要业务操作必须记录 audit_log。
9. 修改代码前先说明计划。
10. 每次任务结束后输出：
   - 修改了哪些文件
   - 为什么这样改
   - 我作为 Java 新手应该理解哪些知识点
   - 如何测试
   - 后续风险
